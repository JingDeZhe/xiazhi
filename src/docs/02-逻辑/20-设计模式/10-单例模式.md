# 单例模式

单例模式是确保一个类只有唯一实例的一种方式，并提供全局的 API 来获取这个实例，在 JS 中可以使用构造函数返回值的特性来生成单例模式的类：

<<<assets/singleton.js

`Singleton`类使用方法和寻常类没有区别，不过只会初始化一次：

```js
const a = new Singleton('mouse')
const b = new Singleton('cat')

console.log(a, b, a === b) // a.name === b.name, a === b
```
