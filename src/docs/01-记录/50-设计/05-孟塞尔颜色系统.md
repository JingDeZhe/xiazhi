# 孟塞尔颜色系统

## 简述

孟塞尔颜色系统通过三个属性来描述颜色：

1. 色相(Hue)
2. 明度(Value)
3. 纯度(Chroma)

此三者的关系如图所示:

![孟塞尔颜色系统](./assets/435px-Munsell-system.svg.png)

更详细的资料请参考以下链接：

1. [维基百科](https://en.wikipedia.org/wiki/Munsell_color_system)
2. [官网](https://munsell.com/)
